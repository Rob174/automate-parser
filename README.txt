Lancer le parser:
- avoir un fichier texte contenant une grammaire
- lancer le Makefile
- entrer le nom du fichier contenant la grammaire puis le nom que l'on souhaite donner au parser
- lancer le fichier .c généré (il a le nom donné au parser à l'etape précedente) avec une chaine de caractères en paramètre
- le programme renvoie OK ou KO selon si la grammaire spécifiée est capable de générer la chaine de caractères
